## 归档规则

1. 活跃文档被新版本取代时，旧版复制到 `archive/`，文件名格式：
   ```
   YYYY-MM-DD-<topic>-<stage>-vX.Y.md
   ```
2. 归档文件保留原始内容（含旧版状态头部），不修改
3. 旧版活跃文档文件名（`<topic>.md`）由新版本接管
4. Plugin 原始输出未经评审的，先存放于此或 `.claude/outputs/`
