## 1. 产品定位

SchemaPlexAI 是企业级 AI 研发协作平台，覆盖从需求定义到交付运营的全生命周期，核心能力包括：

- **AI Agent 执行引擎**: 状态机驱动的智能体编排
- **Spec 规范中心**: 需求文档的版本管理与合规追踪
- **AI 工作流**: BPMN + AI 节点的混合编排
- **RAG 知识中心**: 企业知识库的向量化检索
- **质量门禁**: 自动化的 Spec 偏离检测与合规检查

## 2. 用户角色

| 角色 | 职责 | 核心功能 |
|------|------|----------|
| **平台管理员** | 租户/用户/模型管理 | 系统治理、模型供应商配置 |
| **Agent 开发者** | 创建/配置 Agent | Agent 定义、工具注册、Prompt 设计 |
| **业务用户** | 使用 Agent 完成任务 | Agent 执行、对话交互、结果查看 |
| **质量审核员** | 审核 Agent 输出合规性 | Spec 评审、质量门禁、偏离检测 |
| **运维人员** | 监控与成本管理 | 成本分析、通知管理、系统监控 |

## 3. 功能需求

### 3.1 系统治理（P1）

#### FR-SYS-001 多租户管理
- **描述**: 平台支持多租户隔离，每个租户拥有独立的数据空间
- **验收标准**:
  - 租户创建后自动生成默认管理员账号
  - 所有业务数据自动携带 `tenant_id`
  - 跨租户数据访问被阻断

#### FR-SYS-002 用户与权限
- **描述**: RBAC 权限模型，支持角色、权限、菜单的动态配置
- **验收标准**:
  - 支持自定义角色
  - 权限粒度到按钮级
  - 菜单权限与前端路由联动

#### FR-SYS-003 AI 模型供应商管理
- **描述**: 管理多个 LLM 供应商（OpenAI、Anthropic、Azure 等）
- **验收标准**:
  - 支持供应商增删改查
  - 供应商冷却期自动管理
  - 模型路由与自动降级

### 3.2 Agent 执行引擎（P2）

#### FR-AGE-001 Agent 创建与配置
- **描述**: 可视化配置 Agent 的行为参数
- **配置项**:
  - 模型选择（含供应商）
  - System Prompt（四层注入）
  - Token 预算上限
  - 工具绑定
  - 执行模式（同步/异步）

#### FR-AGE-002 对话式执行
- **描述**: 用户通过自然语言与 Agent 交互
- **验收标准**:
  - 支持流式输出（SSE）
  - 支持执行暂停/恢复/取消
  - 支持上下文记忆（L1 Redis + L2 PG）
  - 工具执行结果实时展示

#### FR-AGE-003 执行记录
- **描述**: 完整的执行历史追踪
- **验收标准**:
  - 记录每次执行的完整状态转换
  - 支持按时间/Agent/状态筛选
  - 支持重新执行历史对话

### 3.3 上下文与 RAG（P3）

#### FR-CTX-001 知识文档管理
- **描述**: 上传、版本管理、删除企业文档
- **支持格式**: PDF, DOCX, TXT, MD, HTML
- **验收标准**:
  - 大文件分片上传
  - 版本历史与回滚
  - 文档状态实时可见（PENDING → PROCESSING → INDEXED）

#### FR-CTX-002 向量检索
- **描述**: 基于语义相似度的知识检索
- **验收标准**:
  - 支持元数据过滤（租户、文档类型、标签）
  - TopK 可配置
  - 检索结果展示相似度分数

### 3.4 Spec 规范中心（P4）

#### FR-SPC-001 Spec 文档管理
- **描述**: 结构化需求文档的编写与管理
- **验收标准**:
  - 支持 Markdown 编辑
  - 版本发布/回滚/对比
  - Steering 文档（Spec 子类型）

#### FR-SPC-002 评审与审批
- **描述**: Spec 变更的评审工作流
- **验收标准**:
  - 支持多人评审
  - 评审意见可线程化讨论
  - 审批通过后锁定变更

### 3.5 质量门禁（P5）

#### FR-QLT-001 执行质量检查
- **描述**: Agent 执行输出自动过质量门禁
- **检查项**:
  - Spec 合规性（输出是否满足 Spec 要求）
  - 输出安全（敏感信息检测）
  - Grounding 验证（事实准确性）
- **验收标准**:
  - 不合规时自动暂停/告警
  - 质量报告可查看

## 4. 非功能需求

### 4.1 性能

| 指标 | 目标 |
|------|------|
| 页面首屏加载 | < 2s |
| API 响应延迟 | P99 < 500ms |
| SSE 首包延迟 | < 1s |
| 向量检索延迟 | P99 < 100ms |

### 4.2 安全

- 所有 API 必须 JWT 认证
- 敏感操作需二次确认
- Token 存储在 sessionStorage（非 localStorage）
- SSE 连接必须携带有效 Token

### 4.3 可用性

- 核心服务 SLA: 99.9%
- 支持优雅降级（模型不可用自动切换）
- 数据备份: 每日全量 + 实时增量
