# Requirements / 需求文档

产品需求文档（PRD）存放目录。

## 范围

- 用户故事（User Stories）
- 功能需求规格
- 非功能需求（性能、安全、可用性）
- 业务流程图

## 命名规范

```
<topic>.md
```

`topic` 为短横线分隔的英文小写主题词，与对应 `specs/`、`designs/`、`plans/`、`ui/` 文档保持一致。
