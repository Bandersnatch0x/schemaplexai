# Standards / 开发规范与流程

开发规范定义项目的编码标准、工作流程和协作规则。所有团队成员必须遵循。

## 范围

- 编码规范（命名、格式、结构）
- 开发工作流程（Feature、Fix、Release）
- 评审标准与检查单
- 文档与 Wiki 约束
- Git 分支策略与合并规范

## 命名规范

```
YYYY-MM-DD[-vX.Y]-<topic>.md
```

`topic` 为短横线分隔的英文小写主题词。
