## 5. 需要人工评审的设计决策

| 决策点 | 选项 A | 选项 B | 建议 |
|--------|--------|--------|------|
| **API Key 展示** | 完全脱敏（********） | 部分显示（sk-abc****xyz） | **待评审**: 部分显示便于识别 |
| **模型配置校验** | 保存时校验 | 实时校验（测试连接按钮） | **待评审**: 实时校验更友好 |
| **租户配额编辑** | 即时生效 | 下次计费周期生效 | **待评审**: 即时生效更直观 |

## 6. API 依赖

```
GET    /api/system/configs              → 系统配置列表
PUT    /api/system/configs              → 更新配置
GET    /api/system/models               → 模型供应商列表
POST   /api/system/models               → 添加供应商
PUT    /api/system/models/{id}          → 更新供应商
DELETE /api/system/models/{id}          → 删除供应商
GET    /api/system/tenants              → 租户列表（管理员）
POST   /api/system/tenants              → 创建租户
PUT    /api/system/tenants/{id}         → 更新租户
```
