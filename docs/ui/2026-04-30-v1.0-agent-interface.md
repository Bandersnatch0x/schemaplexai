> **主题**: `agent-interface`
> **阶段**: `ui`
> **版本**: v1.0

## 1. 页面清单

| 页面 | 路由 | 权限 | 说明 |
|------|------|------|------|
| Agent 列表 | `/agents` | 所有用户 | 浏览、搜索、筛选 Agent |
| Agent 创建/编辑 | `/agents/create` `/agents/:id/edit` | 管理员 | 配置 Agent 参数 |
| Agent 执行器 | `/agents/:id/execute` | 所有用户 | 对话式执行界面 |
| 执行记录 | `/executions` | 所有用户 | 历史执行记录 |
| 执行详情 | `/executions/:id` | 所有用户 | 单条执行详情 |

## 2. Agent 列表页 (`/agents`)

### 布局

```
┌─────────────────────────────────────────────────────────────┐
│  Sidebar  │  Header: Agent 管理                             │
│           ├─────────────────────────────────────────────────┤
│           │  [搜索框] [筛选: 状态 ▼] [创建 Agent +]          │
│           ├─────────────────────────────────────────────────┤
│           │  ┌─────────────┐ ┌─────────────┐ ┌───────────┐ │
│           │  │ Agent Card  │ │ Agent Card  │ │  ...      │ │
│           │  │ [icon]      │ │ [icon]      │ │           │ │
│           │  │ Name        │ │ Name        │ │           │ │
│           │  │ Model: GPT4 │ │ Model: Claude│ │           │ │
│           │  │ Status: ●   │ │ Status: ●   │ │           │ │
│           │  │ [执行] [编辑]│ │ [执行] [编辑]│ │           │ │
│           │  └─────────────┘ └─────────────┘ └───────────┘ │
│           │                      [1] [2] [3] ...            │
└─────────────────────────────────────────────────────────────┘
```

### 组件

- **AgentCard**: 展示 Agent 基本信息，含状态指示器
- **AgentFilter**: 按模型、状态、标签筛选
- **AgentSearch**: 实时搜索（debounce 300ms）

### 交互

- 点击卡片 → 进入执行器
- 点击编辑 → 进入编辑页
- 创建 Agent → 弹窗/新页面

## 3. Agent 执行器 (`/agents/:id/execute`)

### 布局

```
┌─────────────────────────────────────────────────────────────┐
│  Sidebar  │  Header: Agent 名称 + 状态                       │
│           ├─────────────────────────────────────────────────┤
│           │  ┌─────────────────────────────────────────────┐│
│           │  │  ChatHistory                                ││
│           │  │  ┌──────────┐                               ││
│           │  │  │ User     │ 输入内容                       ││
│           │  │  └──────────┘                               ││
│           │  │  ┌──────────────┐                           ││
│           │  │  │ Assistant    │ 回复内容                   ││
│           │  │  │ [思考中...]  │                           ││
│           │  │  └──────────────┘                           ││
│           │  │  ┌──────────────┐                           ││
│           │  │  │ Tool Call    │ 🔧 执行了 git status      ││
│           │  │  └──────────────┘                           ││
│           │  └─────────────────────────────────────────────┘│
│           ├─────────────────────────────────────────────────┤
│           │  [输入框...]  [发送] [暂停 ■] [取消 ×]           │
│           │  Token: 1,234 / 8,000 | 预算剩余: 85%           │
└─────────────────────────────────────────────────────────────┘
```

### 组件

- **ChatHistory**: 滚动加载历史消息，区分用户/助手/工具消息
- **ChatInput**: 多行输入框，支持 Shift+Enter 换行
- **TokenBudgetBar**: 底部显示 Token 使用情况
- **SseViewer**: 实时流式输出，支持打字机效果

### 状态流转

```
Idle → Typing → Sending → Streaming → Thinking → ToolCalling → Completed
                                      ↓
                                  Paused (用户暂停)
```

### SSE 连接管理

- 使用 `fetch + ReadableStream`（非 `EventSource`，以支持 Header 携带 Token）
- 组件卸载时自动 `AbortController.abort()`
- 最大重连次数: 3

## 4. 执行记录页 (`/executions`)

### 表格字段

| 列 | 说明 |
|----|------|
| ID | 执行 ID，可点击跳转详情 |
| Agent | Agent 名称 |
| 状态 | Tag 颜色: 成功(green) / 失败(red) / 暂停(orange) / 运行中(blue) |
| Token 消耗 | input + output |
| 耗时 | 开始 → 结束 |
| 操作 | 查看详情 / 重新执行 |

### 筛选

- 按时间范围（今天/本周/本月）
- 按状态
- 按 Agent

## 5. 设计规范

### 色彩

```css
--primary: #1677ff;          /* Ant Design 主色 */
--success: #52c41a;
--warning: #faad14;
--error: #f5222d;
--info: #1890ff;
--streaming: #722ed1;        /* 流式输出指示 */
```

### 间距

- 页面内边距: 24px
- 卡片间距: 16px
- 表单项间距: 16px

### 响应式

- ≥ 1440px: 三列卡片布局
- ≥ 1024px: 两列卡片布局
- < 1024px: 单列布局，侧边栏收起为图标
