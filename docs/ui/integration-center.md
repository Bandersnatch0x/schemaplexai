## 5. 需要人工评审的设计决策

| 决策点 | 选项 A | 选项 B | 建议 |
|--------|--------|--------|------|
| **Git OAuth 后仓库授权** | 全量授权 | 按需勾选 | **待评审**: 按需勾选更安全 |
| **MCP 工具展示** | 列表 | 卡片 | **待评审**: 卡片适合展示工具详情 |
| **集成调用日志保留** | 7 天 | 30 天 | **待评审**: 30 天便于问题排查 |

## 6. API 依赖

```
GET    /api/integration/git/connections   → Git 连接状态
POST   /api/integration/git/connect       → OAuth 连接
GET    /api/integration/git/repos         → 仓库列表
POST   /api/integration/mcp/servers       → 注册 MCP Server
POST   /api/integration/mcp/servers/{id}/connect    → 连接
POST   /api/integration/mcp/servers/{id}/disconnect → 断开
GET    /api/integration/mcp/servers/{id}/tools     → 工具列表
```
