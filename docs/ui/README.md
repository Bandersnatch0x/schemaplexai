# UI / UI/UX 设计文档

UI/UX 设计文档存放目录。每份文档对应一个前端主题，描述页面结构、组件拆分、交互流程与视觉规范。

## 范围

- 页面结构与布局（Wireframe / Layout）
- 组件拆分与复用策略
- 交互流程与状态转换
- 视觉规范（色彩、字体、间距）
- 响应式与可访问性（a11y）要求

## 命名规范

```
<topic>.md
```

`topic` 与对应主题的 `docs/designs/`、`docs/specs/`、`docs/plans/` 文档保持一致。
