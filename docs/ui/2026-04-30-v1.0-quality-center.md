## 5. 需要人工评审的设计决策

| 决策点 | 选项 A | 选项 B | 建议 |
|--------|--------|--------|------|
| **规则严重等级图标** | 颜色区分 | 图标 + 颜色 | **待评审**: 色盲友好考虑 |
| **误报处理** | 全局白名单 | 按规则白名单 | **待评审**: 按规则更精细 |
| **报告导出** | PDF | JSON / CSV | **待评审**: PDF 适合人工阅读，JSON 适合系统对接 |

## 6. API 依赖

```
GET    /api/quality/gates              → 门禁配置列表
POST   /api/quality/gates              → 创建规则
PUT    /api/quality/gates/{id}         → 更新规则
DELETE /api/quality/gates/{id}         → 删除规则
GET    /api/quality/reports            → 质量报告列表
GET    /api/quality/reports/{executionId} → 执行质量详情
POST   /api/quality/reports/{id}/resolve → 处理告警
```
