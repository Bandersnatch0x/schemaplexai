推荐方案

Inter
Inter（英文/数字/标题）
由 Rasmus Andersson 设计，专为屏幕 UI 优化。开源（SIL OFL），Google Fonts 免费提供。字形干净、字重丰富（100-900），与 macOS/iOS 系统字体几乎无法区分，是 x.com 风格的最佳开源替代。
SIL Open Font License Google Fonts 9 字重
中文
Noto
Noto Sans SC（中文回退）
Google 与 Adobe 联合开发的泛用中文字体。开源（SIL OFL），覆盖简体中文、繁体中文、日文、韩文。字形中性现代，与 Inter 搭配视觉协调，无风格冲突。
SIL Open Font License Google Fonts 7 字重
007
JetBrains
JetBrains Mono（数据/代码）
JetBrains 开发的开源等宽字体。专为代码设计，但用于数据展示同样出色。数字等高宽、零带斜线区分、大写 I 和小写 l 易区分。开源（SIL OFL）。
SIL Open Font License Google Fonts 等宽

字体栈配置

// CSS font-family 栈
--font-sans: 'Inter', 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', sans-serif;
--font-mono: 'JetBrains Mono', 'Fira Code', 'SF Mono', monospace;
// Tailwind / CSS 变量映射
--font-title: var(--font-sans); // 字重 600
--font-body: var(--font-sans); // 字重 400
--font-data: var(--font-mono); // 字重 500
--font-input: var(--font-sans); // 字重 400, 15px
