关键发现：图中背景色比我的方案偏蓝（#0a0e1a vs #050507），这赋予了更明显的"科技/太空"感而非纯粹的深渊黑。 卡片底使用了 #111827（Tailwind gray-900），与背景的对比度更温和。建议采纳图中的蓝黑背景，它更契合黑镜的冷峻科技感。

提取 2：导航设计（基于 Task Timeline + Cell Detail）

渐进式 Sidebar
  • 宽度约 200px，背景 #0a0e1a
  • 顶部 Logo 区（小六边形图标 + 系统名）
  • 导航项：图标(16px) + 文字，间距 12px
  • 选中态：左侧 3px 荧光竖线 + 背景 #111827
  • 未选中态：文字 #64748b，图标同色
  • 分组标题：全大写 + 字间距，更小的字号
  • 底部用户区：头像 + 名称 + 角色
沉浸式 Sidebar
  • 宽度压缩至 52px，仅显示图标
  • 图标容器：36×36px 圆角 8px
  • 选中态：图标容器背景 #00d4aa20 + 荧光边框
  • 未选中态：图标容器背景 #1e2a33
  • hover 展开：tooltip 显示文字
  • 底部设置/用户图标常驻

提取 4：右侧面板设计（基于 Task Hive.png）

选中节点后的属性面板结构
  • 面板宽度：320px，从右侧滑入（slide-in），overlay 模式（不挤压画布）
  • 头部：节点名称 + 六边形图标 + 关闭按钮（×）
  • 标签页：Properties / Logs / Metrics（药丸切换或底线切换）
  • 属性区：表单字段（key-value 对），label 左对齐，value 右对齐
  • 操作区>：底部固定，Run / Stop / Edit 按钮
  • 背景：半透明遮罩 + 面板背景 #0a0e1a + 左边框荧光

提取 5：甘特图设计（基于 Task Timeline.png）

时间轴
  • 顶部时间刻度：小时/分钟，#64748b 小字号
  • 垂直网格线：#1e2a33 半透明
  • 当前时间指示：荧光竖线 + 脉冲动画
任务段（Bars）
  • 高度：20-24px，圆角 4px
  • 颜色编码：青=成功、琥珀=执行中、红=失败、紫=等待
  • 段内文字：任务名，#e2e8f0，9-10px
  • hover：段高度微增 + 阴影光晕
行标题
  • 工作流名称 + 状态圆点
  • 固定宽度（不随滚动移动）
  • hover 行高亮

  1. 背景色调整：深渊黑 → #0a0e1a（蓝黑），更有科技冷峻感
  2. 卡片底调整：蜂房卡片 → #111827（与背景对比更柔和）
