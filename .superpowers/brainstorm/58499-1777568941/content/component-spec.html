组件规范：Ant Design 覆写策略

基于参考图"深渊蜂巢"的融合方案，保持 AntD 结构，覆写视觉层

设计原则

  • 结构保留：AntD 组件的 DOM 结构、交互逻辑、API 不变
  • 视觉覆写：通过 ConfigProvider theme token + 全局 CSS 变量 + 少量自定义 class 实现风格转换
  • 无边框化：尽可能弱化边框，用背景色差异和微间距建立层次
